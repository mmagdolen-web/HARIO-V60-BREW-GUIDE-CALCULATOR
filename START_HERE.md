## Quick Deploy Path (5 Minutes) 🚀

### 1️⃣ Create GitHub Repo
- Go to [github.com](https://github.com)
- Click **+** → **New repository**
- Name: `brew-calculator`
- Check **Public** ✅
- Check **Add a README file** ✅
- Click **Create repository**

### 2️⃣ Upload Files
- Click **Add file** → **Upload files**
- Drag & drop ALL files from this folder
- Click **Commit changes**

**Important**: Folder structure must be:
```
root/
├── index.html
└── src/lib/brewingLogic.js
```

### 3️⃣ Enable GitHub Pages
- Click **Settings** (top right)
- Click **Pages** (left sidebar)
- Set: Source = "Deploy from a branch"
- Set: Branch = "main" + "/ (root)"
- Click **Save**
- ⏳ Wait 2-3 minutes...
- ✅ Green checkmark = Live!

### 4️⃣ Test Your App
Visit: `https://YOUR_USERNAME.github.io/brew-calculator`

### 5️⃣ Add to iPhone
- Open Safari on iPhone
- Visit your live URL above
- Tap **Share** → **Add to Home Screen**
- Tap **Add**
- ✅ App now on home screen!

## Troubleshooting 🔧

**Blank page after upload?**
- Hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
- Check console (F12 → Console) for errors

**"Page not found"?**
- Verify GitHub Pages is enabled (Settings → Pages)
- Check branch is "main" and folder is "/ (root)"
- Wait a few minutes if you just enabled Pages

**Folder structure looks wrong?**
- Ensure: `index.html` in root
- Ensure: `src/lib/brewingLogic.js` in src/lib/ (not root!)
- Upload src/ folder, not individual files

**iPhone app won't launch?**
- Try different browser (Chrome, Firefox)
- Delete app from home screen
- Add again (Share → Add to Home Screen)
